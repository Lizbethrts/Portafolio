# Portafolio Astro

Proyecto base para un portafolio personal creado con [Astro](https://astro.build/).

## Comandos

```sh
npm install
npm run dev
npm run build
npm run preview
```

## Personalizacion rapida

- Edita `src/pages/index.astro` para cambiar tu nombre, descripcion, proyectos y enlaces.
- Edita `src/layouts/Layout.astro` para cambiar el titulo y la descripcion SEO.
- Reemplaza los enlaces de contacto por tu correo, GitHub y LinkedIn.

## Subir a GitHub

```sh
git add .
git commit -m "Crear portafolio Astro"
git branch -M main
git remote add origin https://github.com/tu-usuario/portafolio-astro.git
git push -u origin main
```
